Le langage de balisage hypertexte (HTML) : Ce que c'est et comment ça fonctionne

Qu'est-ce que le HyperText Markup Language (HTML) ?

Le HyperText Markup Language (HTML) est un ensemble de symboles ou de codes de balisage insérés dans un fichier destiné à être affiché sur internet. Le balisage indique aux navigateurs web comment afficher les mots et les images d'une page web. Chaque morceau individuel de code de balisage (qui se trouve entre les caractères « < » et « > ») est appelé un élément, bien que beaucoup de gens l'appellent également une balise. Certains éléments viennent en paires, indiquant quand un effet d'affichage doit commencer et quand il doit se terminer.

Comprendre HTML

Le HyperText Markup Language est le langage informatique qui facilite la création de sites web. Le langage, qui a des mots de code et une syntaxe comme tout autre langage, est relativement facile à comprendre et, avec le temps, devient de plus en plus puissant dans ce qu'il permet de créer. HTML continue d'évoluer pour répondre aux demandes et aux exigences de l'internet sous la direction du World Wide Web Consortium, l'organisation qui conçoit et maintient le langage—par exemple, avec la transition vers le Web 2.0.
L'HyperText est la méthode par laquelle les utilisateurs d'internet naviguent sur le web. En cliquant sur un texte spécial appelé hyperliens, les utilisateurs sont amenés vers de nouvelles pages. L'utilisation de "hyper" signifie qu'il n'est pas linéaire, donc les utilisateurs peuvent aller n'importe où sur internet simplement en cliquant sur les liens disponibles.
Le balisage est ce que les balises HTML font au texte à l'intérieur d'elles ; elles le marquent comme un type spécifique de texte. Par exemple, un texte de balisage pourrait être en gras ou en italique pour attirer l'attention sur un mot ou une phrase spécifique.

Bases du HTML

Bases du HTML À la base, HTML est une série de codes courts tapés dans un fichier texte. Ce sont les balises qui alimentent les capacités du HTML. Le texte est enregistré en tant que fichier HTML et affiché via un navigateur web. Le navigateur lit le fichier et traduit le texte en une forme visible, conformément aux codes utilisés par l'auteur pour écrire ce qui devient le rendu visible. L'écriture HTML nécessite que les balises soient utilisées correctement pour créer la vision de l'auteur.
Les balises sont ce qui sépare le texte normal du code HTML. Les balises sont les mots entre ce que l'on appelle les crochets angulaires, qui permettent aux graphiques, images et tableaux d'apparaître sur la page web. Différentes balises remplissent différentes fonctions. Les balises les plus basiques appliquent une mise en forme au texte. Alors que les interfaces web doivent devenir plus dynamiques, les feuilles de style en cascade (CSS) et les applications JavaScript peuvent être utilisées. CSS rend les pages web plus accessibles, et JavaScript ajoute de la puissance au HTML de base
